# Complete command order for openspec

install openspec : npm install -g @fission-ai/openspec@latest

init openspec in teh project : openspec init

we need to add the project info in this file (one time in the begining of the project ): openspec/config.yaml (like agent.md)


```phase 1 (propose the feature )
/opsx-propose add-receipts-crud-foundation


```phase 2 (apply the changes)
/opsx-apply add-receipts-crud-foundation


```phase 3 ( sync the changes )
/opsx-sync add-receipts-crud-foundation


```phase 4 (archive the changes )
/opsx-archive add-receipts-crud-foundation
